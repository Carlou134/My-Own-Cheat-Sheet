Tablas HTML

Etiquetas Nombre Descripción
<table> Table Crear la tabla

<th> Table Headings Titulos de la tabla
<td> Table Data Cell Celdas que contienen información
<thead> Table Head Encabezado de la tabla
<tbody> Table Body Cuerpo de la tabla
<tfoot> Table Footer La parte inferior de la tabla
